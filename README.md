The **Word Guessing Game** is a simple web-based game developed using **HTML, CSS, and JavaScript**. In this game, the player has to guess a hidden word letter by letter within a limited number of attempts.

## ğŸ•¹ How the Game Works

1. A random word is selected from a predefined list
2. A hint related to the word is displayed
3. The player types one letter at a time
4. Correct letters are displayed in the word boxes
5. Incorrect letters reduce the remaining guesses
6. The game ends when:

   * The player guesses the full word correctly (Win)
   * The player runs out of guesses (Game Over)
